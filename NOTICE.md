# Notice

This APV2.1 repository is a temporary private review package. It is shared so invited reviewers can inspect the current prototype, run local tests, and understand the research direction.

It is not an open-source release. The code, documentation, generated reports, experiment traces, and skill experience packages are protected by the temporary review license in [LICENSE](LICENSE).

The project contains provenance metadata and release-lineage markers used to preserve authorship continuity and detect unauthorized code reuse. These markers are ordinary metadata and validation helpers; they do not alter AP cognition, do not watermark generated outputs, and do not execute hidden behavior.

The main research direction is called **Endogenous Cognitivism / 内生认知主义**: cognition is treated as an endogenous information-flow closure among perception, state-field competition, attention, memory, cognitive feeling, emotion, action, and feedback.
